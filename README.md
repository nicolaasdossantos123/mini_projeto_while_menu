# Mini Projeto — Menu Interativo com while (Python)

Este projeto é um mini sistema em Python criado para praticar **lógica de programação**, utilizando principalmente o laço de repetição `while`.

##  Objetivo
Praticar:
- Laços de repetição (`while`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Entrada de dados com `input()`
- Contadores e operadores lógicos
- Organização de um programa com menu

##  Funcionalidades
O programa apresenta um menu interativo com as seguintes opções:

1️⃣ Somar dois números  
2️⃣ Contar números positivos  
3️⃣ Contar números pares  
4️⃣ Contar números ímpares  
0️⃣ Sair do programa  

O menu permanece ativo até o usuário escolher a opção **0**.

##  Tecnologias utilizadas
- Python 3

##  Como executar o projeto
1. Certifique-se de ter o Python instalado
2. Abra o terminal na pasta do projeto
3. Execute o comando:

```bash
python mini_projeto_menu.py
